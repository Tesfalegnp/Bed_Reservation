#Bed_reservation
A CRUD application for managing bed reservations.

#Tesfalegn Petros
##Tutorial Bed Reservation CRUD Laravel 10
      ###Features
Reserve Bed

Cancel Reservation

View Available Beds

Manage Reservations

CRUD Laravel 10

Validation

SweetAlert Notifications

Datatable for Listings

JavaScript Validation

FontAwesome 6 Icons

    ###Required
Laravel 10

PHP (latest version)

Composer

Datatable

SweetAlert

JavaScript

FontAwesome 6

Server (XAMPP/Laragon)

  ##Installation
Download or clone the source code. Create a database named bed_reservation.

Install the dependencies and start the server:

composer update
php artisan migrate:fresh
php artisan serve
Open the link in your browser: http://127.0.0.1:8000
